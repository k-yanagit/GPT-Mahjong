# 「++」以下の情報は以下のとおりです
- 左から**：**で区切って萬子、索子、筒子、字牌です。
- アガリ形は上がった時の手牌です。
- ツモ/ロンで示される牌が、上がり牌です。

- 局、親/子、ドラ、リーチのTrue/Falesは点数計算に必要な情報です。

# 「++」以下にあるアガリの形を見て以下を決定してください。
- 翻数
- 符
- 役満か否か
- 親、子が払う点数
